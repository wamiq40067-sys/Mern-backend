# MERN Stack Backend Learning

This repository contains my backend development practice work completed during my MERN Stack certification journey.

It focuses on building server-side applications using Node.js, Express.js, and MongoDB.

---

## About This Project

This backend project is part of my learning process in full-stack MERN development.  
It includes hands-on practice with APIs, database operations, authentication, and server-side logic.

---

## Technologies Used

- Node.js
- Express.js
- MongoDB
- Mongoose
- REST APIs
- Postman (for API testing)

---

## Key Backend Concepts Covered

### 1. Server Setup
- Creating a Node.js server
- Setting up Express application
- Handling routes and requests

### 2. REST API Development
- GET, POST, PUT, DELETE methods
- Building CRUD operations
- Structuring API endpoints

### 3. Database Integration
- Connecting MongoDB with Mongoose
- Creating schemas and models
- Performing database operations

### 4. Middleware Usage
- Express middleware setup
- Request parsing (JSON handling)
- Error handling middleware

### 5. Authentication Basics
- User registration and login flow
- Password handling (basic understanding)
- Token-based authentication (conceptual learning)
